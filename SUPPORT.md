# Support

MeetYou is currently maintained as an active development project.

Use GitHub Issues for:

- Reproducible bugs.
- Documentation gaps.
- Installation or setup problems.
- Feature proposals.
- Provider integration questions.

Before opening an issue:

- Search existing issues and docs.
- Include your operating system, Python version, Node.js version, and relevant command output.
- Redact tokens, cookies, credentials, private hostnames, chat IDs, local usernames, and personal file paths.

For suspected vulnerabilities, use [SECURITY.md](./SECURITY.md) instead of a public issue.
